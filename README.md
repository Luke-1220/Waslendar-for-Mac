## Waslendar-for-Mac
画面右上のメニューバーにカレンダーのアイコンが表示され、Mac標準カレンダーの予定を簡単に確認できます。
<!--more-->
アイコンを押すだけでスケジュールを確認することができるので、ビデオ会議中に突然「明日の15時から空いてる？」と言われても困ることはありません。

メニューバーに表示する内容は、アイコン、「次の予定まであと何分」、「時計」など色々選べます。
